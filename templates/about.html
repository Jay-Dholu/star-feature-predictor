Star Feature Predictor

The Star Feature Predictor is a machine learning-based web application designed to predict missing stellar characteristics such as:

  • 🌡️ Temperature
  • ☀️ Luminosity
  • 🪐 Radius
  • ⚖️ Mass
  • ✨ Star Type
  • 🔢 Absolute Magnitude

This model was developed as a personal project to practice machine learning techniques using astronomical data.

It predicts missing star features like temperature, radius, luminosity, and type based on known values, showcasing the power of data-driven inference in astrophysics.

Perfect for experimenting with regression, classification, and feature prediction in a cosmic context.
